# 第1章 はじめに

本章では、技術書執筆テンプレートの基本的な使い方を説明します。

## コードブロックの例

プログラムコードは以下のように記述します。

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

Python の例:

```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 120
```

## 画像の埋め込み

画像は以下のように埋め込みます。

```markdown
![回路図の説明](../assets/diagrams/circuit.svg)
```

実際のプロジェクトでは、`src/assets/images/` または `src/assets/diagrams/` に画像を配置してください。
